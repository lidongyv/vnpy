## 🔖 核心概念速查

### VNPy核心组件

| 组件 | 说明 | 详见文档 |
|-----|------|---------|
| **MainEngine** | 主引擎，管理所有模块 | 使用指南 3.2.1 |
| **EventEngine** | 事件驱动引擎 | 使用指南 3.2.2 |
| **Gateway** | 交易接口 | 使用指南 3.2.3 |
| **CtaTemplate** | CTA策略模板 | 使用指南 7.1 |
| **AlphaDataset** | 因子数据集 | 使用指南 5.3 |
| **AlphaModel** | ML模型 | 使用指南 5.4 |

### 核心算子分类

| 类别 | 说明 | 典型算子 | 详见文档 |
|-----|------|---------|---------|
| **时序(TS)** | 单标的时间序列 | ts_mean, ts_std, ts_corr | 参考手册 |
| **截面(CS)** | 多标的横向比较 | cs_rank, cs_scale | 参考手册 |
| **数学** | 数学运算 | log, abs, quesval | 参考手册 |
| **技术(TA)** | 技术指标 | ta_rsi, ta_atr | 参考手册 |
| **AlphaGPT核心** 🆕 | 高级量化算子 | gate, jump, decay | AlphaGPT参考 |
| **MemeIndicators** 🆕 | 专业因子指标 | liquidity_health, fomo_acceleration | AlphaGPT参考 |

### 典型因子公式

**VNPy原生因子**:
```python
# 动量因子
momentum = "close / ts_delay(close, 20) - 1"

# 波动率因子
volatility = "ts_std(close / ts_delay(close, 1) - 1, 20)"

# 量价相关
vol_price_corr = "ts_corr(close, volume, 20)"

# Alpha因子
alpha1 = "cs_rank(ts_argmax(pow1(close, 2), 5)) - 0.5"
```

**AlphaGPT增强因子** 🆕:
```python
# 条件过滤
safe_signal = "gate(liquidity_health(liquidity, fdv) - 0.5, buy_signal, 0)"

# 异常检测
crash_detect = "jump(close / ts_delay(close, 1) - 1, 3.0)"

# FOMO情绪
fomo_signal = "fomo_acceleration(volume) * buy_sell_imbalance(close, open, high, low)"

# 完整策略
meme_strategy = "gate(safety, entry * (1 - pump_deviation(close, 20)), 0)"
```

详见《量化算子快速参考手册》和《AlphaGPT算子快速参考》的因子组合章节。
